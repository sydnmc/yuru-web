api

endpoints

/gds - returns the osu! guest difficulties list for lilac or sydney, requires person=STRING option

/sets - returns the list of osu! beatmapsets that both of us have done

/songInfo - using last.fm, returns the song last listened to, or the song we're currently listening to

/pkInfo - using pluralkit, frontList=true returns the last fronts for the number of days specified by before=INT, as well as a percentage that each person has fronted in that timeframe. without the frontList option, it requires a user=STRING option to fetch a user by their pluralkit id
