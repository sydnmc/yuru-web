api

ꕤ endpoints

/gds - returns the osu! guest difficulties list for lilac or sydney, requires person option

/sets - returns the list of osu! beatmapsets that both of us have done

/lastfm - using last.fm, returns the song last listened to, or the song we're currently listening to

/recentFronts - using pluralkit, returns information about the last front each alter had, as well as a percentage that each alter has fronted in that timeframe

/frontData - using pluralkit, returns historical front information for our system

/newMapData - allows you to see the basic info about an osu! map in the format that they're servered on our website. requires beatmapId or beatmapsetId options

/changeInfo - accepts new set or gd information. requires a yuru.ca api access token
